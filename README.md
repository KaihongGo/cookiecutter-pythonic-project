## 使用步骤

### 1. 安装 Cookiecutter

```shell script
pip install cookiecutter
```

### 2. 创建项目

```shell script
cookiecutter https://github.com/pyloong/cookiecutter-pythonic-project
```

根据提示输入项目基础信息配置。
